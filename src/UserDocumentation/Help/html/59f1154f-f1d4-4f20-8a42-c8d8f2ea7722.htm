CalcIVSulator - Code documentation

Welcome to the code documentation of CalcIVSulator. We hope that this documentation will give you peace in mind, hope and understanding of this project.

Getting Started with the documentation

You can find in this documentation all the functions which are used, its parameters and return values. We even documented functions which are used from official Microsoft libraries.

You can simply choose from following sections:

  • CalculatorGUI - Visual GUI for user. You can test on this application our math library. It is simple from application

  • IVSMathLibrary - Core mathematical library. Here you can find all functions which can be used in your application.

  • IVSMathLibraryTests - Test for library which confirms its functions.

Thank you for your interest in our CalcIVSulator and its math library. Feel free to give us feedback or create issue on our github.
